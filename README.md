# 전국 친환경차 현황 대시보드

대한민국의 전기차 보급 현황, 지역별 보조금, 충전소 인프라 데이터를 한눈에 확인하고 비교할 수 있는 페이지입니다.

## 주요 기능

### 1. 전국 보급 현황 
- **지도**: 전국 지도를 통해 지역별 친환경차 보급률을 시각적으로 확인합니다.
- **지역별 추이**: 선택한 지역의 일반 차량 대비 전기차 등록 성장 추이를 그래프로 분석합니다.
- **보조금 현황**: 지자체별/차종별(승용, 화물, 초소형) 보조금 정보를 표 형태로 제공합니다.

### 2. 차량 유지비 비교 
- 전기차와 내연기관차(휘발유/경유)의 연간 주행 거리 및 연료비를 기반으로 실제 경제성을 시뮬레이션합니다.

### 3. 충전소 인프라 현황 
- 전국 충전소 위치 및 운영 기관별 현황을 데이터로 확인하여 인프라 밀집도를 분석합니다.

### 4. 친환경차 통합 FAQ 
- 전기차 구매부터 관리까지 사용자가 궁금해하는 질문들을 카테고리별로 제공합니다.

## 시작하기

상세한 설치 및 실행 방법은 [SETUP_GUIDE.md](./SETUP_GUIDE.md)를 참고하세요.

### 이후...
```bash
# 1. DB 실행
docker-compose up -d

# 2. 데이터 마이그레이션 (추출된 CSV 데이터를 DB에 적재)
python migrate_data.py

# 3. 대시보드 실행
streamlit run app.py
```
