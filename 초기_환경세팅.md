## 🛠️ 2. 파이썬 환경 세팅 및 라이브러리 설치

> ⚠️ **주의사항 (버전 충돌 방지)**
> Python 3.14 같은 최신 버전은 오류가 날 수 있습니다. 가급적 **Python 3.11 또는 3.12 버전** 사용을 권장합니다.

VS Code 등 에디터에서 터미널을 열고 아래 명령어들을 순서대로 입력합니다.

**1) 가상환경 생성 및 실행 (선택사항이지만 권장)**
```bash
# 가상환경 생성 (macOS/Windows 공통)
py -m venv .venv

# 가상환경 실행 (Windows)
.\.venv\Scripts\activate
# 가상환경 실행 (macOS/Linux)
source venv/bin/activate
```

**2) 필수 라이브러리 설치**
프로젝트에 필요한 화면 구성(streamlit), 데이터 처리(pandas), 지도 및 그래프 시각화(plotly) 패키지를 한 번에 설치합니다.

## 📝 3. 초기 뼈대(Skeleton) 코드 작성
생성해둔 3개의 파이썬 파일에 아래 코드를 각각 복사해서 붙여넣고 저장(`Ctrl+S`)해 주세요. 

### 📄 `app.py` (메인 페이지)
```python
import streamlit as st

# 페이지 기본 설정 (가장 위에 와야 함)
st.set_page_config(page_title="친환경 모빌리티 대시보드", page_icon="🌍", layout="wide")

st.title("🗺️ 전국 친환경 모빌리티 보급 현황")
st.write("이곳에는 Plotly를 활용한 전국 색상 지도와 지역별 핵심 요약 지표가 들어갈 예정입니다.")

# 임시 레이아웃 분할
col1, col2 = st.columns([2, 1])
with col1:
    st.info("지도 시각화 영역 (공공데이터 CSV 연동 예정)")
with col2:
    st.success("지역별 세부 그래프 영역")
```

### 📄 `pages/1_Compare.py` (2페이지)
```python
import streamlit as st

st.set_page_config(page_title="유지비 비교 분석", page_icon="⚖️")

st.title("⚖️ 내연기관 vs 친환경차 유지비 비교")
st.write("사용자의 주행 거리에 따른 연료비 및 N년차 누적 유지비를 비교합니다.")

# 임시 슬라이더 UI
mileage = st.slider("연간 예상 주행거리(km)를 선택하세요", 5000, 30000, 15000, step=1000)
st.write(f"선택한 주행거리: **{mileage} km**")

st.warning("이곳에 비교 Bar 차트와 Line 차트가 들어갈 예정입니다.")
```

### 📄 `pages/2_FAQ.py` (3페이지)
```python
import streamlit as st

st.set_page_config(page_title="친환경차 통합 FAQ", page_icon="❓")

st.title("❓ 친환경차 보조금 & 혜택 FAQ")

# 임시 검색창 UI
search_query = st.text_input("궁금한 키워드를 검색해 보세요 (예: 보조금, 충전)")

# 임시 아코디언(Expander) UI
with st.expander("Q. 2026년 전기차 보조금은 얼마인가요?"):
    st.write("A. 지자체별로 상이하며, 서울시 기준 최대 OOO만 원입니다. (데이터 연동 예정)")
```
